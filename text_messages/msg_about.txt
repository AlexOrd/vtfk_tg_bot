ℹ️ <b>Про цього бота</b>

Цей бот — демонстраційний проєкт для майстер-класу ВТФК, написаний на <b>Python</b> з використанням бібліотеки <b>aiogram</b>.

<b>Головна ідея архітектури:</b>
Логіка бота <b>відокремлена</b> від контенту. Це дозволяє змінювати тексти без втручання в код.

<b>Структура проєкту:</b>
<pre>
.
├── bot.py           # Головний мозок — вся логіка тут
├── file_reader.py   # Читає тексти з файлів
├── requirements.txt # Список залежностей
└── text_messages/   # Всі тексти для бота
    ├── msg_start.txt
    ├── msg_greeting.txt
    ├── msg_about.txt
    └── ...
</pre>

<b>Можливості бота:</b>
• 🔘 <b>Базовий режим</b> — реагує на кнопки, відповідає текстом з файлів
• 🧠 <b>Режим ШІ</b> — підключається до OpenAI Assistants API і веде діалог з пам'яттю контексту

<b>Залежності</b> (з <code>requirements.txt</code>):
<pre>
aiogram
openai
python-dotenv
</pre>
