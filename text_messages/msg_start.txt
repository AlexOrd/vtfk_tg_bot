🤖 <b>Вітаємо на майстер-класі ВТФК!</b>

Це — шаблон Telegram-бота, який демонструє, як бот реагує на кнопки та команди, використовуючи заздалегідь підготовлені тексти.

<b>Що вміє цей бот:</b>
• 👋 Привітатись і розповісти про себе
• ℹ️ Показати технічні деталі проєкту
• 🌐 Відкрити офіційний сайт ВТФК
• 🤖 Вести діалог з ШІ-асистентом <i>(якщо налаштовано OpenAI)</i>

<b>Стек технологій:</b>
• 🐍 <b>Python 3</b> — мова програмування
• 📚 <b>aiogram</b> — бібліотека для Telegram Bot API
• 🧠 <b>OpenAI Assistants API</b> — ШІ з пам'яттю контексту
• ☁️ <b>GitHub Codespaces</b> — середовище розробки без налаштувань

Натисніть на кнопку нижче, щоб почати! 👇
